=============
Installation
=============

to get the project do :

.. code-block:: bash

    git clone https://github.com/foxmask/django-th.git

or 

.. code-block:: bash

    pip install django-th

and at least :

.. code-block:: bash

    python manage.py syncdb

to startup the database


Dependencies
------------
* Python 3.4.0
* Django > 1.6
* django-th-rss >= 0.3.0
* django-th-evernote >= 0.3.0
