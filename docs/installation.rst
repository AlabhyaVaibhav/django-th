=============
Installation
=============

To get the project do :

From GitHub 
===========

.. code-block:: bash

    git clone https://github.com/foxmask/django-th.git

From Pypi
=========

.. code-block:: bash

    pip install django-th



Dependencies
------------
* Python 3.4
* Python 2.7
* Django >= 1.7
* django-th-rss >= 0.3.0
* django-th-pocket >= 0.2.0
* django-js-reverse == 0.3.3
